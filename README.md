# 🖥️ hack.PCgame

**あなたのPCがハッカーに乗っ取られた——。**

デスクトップ画面を操作しながら、ハッカーの脅迫に立ち向かう**没入型デスクトップホラーゲーム**です。

> 🎮 **プレイはこちら**: [https://xiaolincangkong5-web.github.io/hack-pcgame/](https://xiaolincangkong5-web.github.io/hack-pcgame/)

---

## 📖 ストーリー

ある日、あなたのPCが突然ハッカーに乗っ取られた。
「面白いファイルを見つけたよ。拡散してほしくなければ、おとなしくしてな。」

デスクトップに現れる不気味なメモ帳のメッセージ。
次々と開かれるアプリケーション。
そして——あなたの秘密が、ネットに晒されようとしている。

**あなたは、この窮地をどう切り抜ける？**

---

## 🎮 ゲームの流れ

1. **Day1** — デスクトップ探索。ファイルを開き、ハッカーの痕跡を探る。
2. **Day2** — ハッキングシーケンス。ウイルスタイマーとの戦い。
3. **Day3** — 最終決断。あなたの選択がエンディングを分ける。

---

## 🏆 エンディング（全7種）

| エンディング | 説明 |
|------------|------|
| 🗑️ 廃棄エンド | PCを窓から投げ捨てる |
| 🔍 真実エンド | ハッカーの正体を暴く |
| 🚔 逮捕エンド | 警察に通報する |
| 💀 ハッキングエンド | ウイルスタイマーがゼロに |
| 🔥 炎上エンド | ネットで拡散される |
| 🌀 破棄エンド | 全てのデータを消去する |
| 🚪 隣人エンド | 隣人に襲われる |

---

## 🎯 実績（全10種）

ゲーム内の特定の条件を満たすと実績が解除されます。
実績画面で進捗を確認できます。

---

## 🛠️ 技術情報

- **言語**: HTML / CSS / JavaScript（フロントエンドのみ）
- **音声**: Web Audio API によるシンセサイズ効果音 + 一部MP3
- **保存**: localStorage（セーブデータ・音量設定・実績）
- **外部依存**: なし（CDN等不使用）

```

├── audio.js       — サウンドマネージャー
├── events.js      — ハッキングシーケンス

├── hallway.js     — 廊下・警察署前
├── endings.js     — エンディング演出

assets/
├── images/        — 画像ファイル
└── se/            — 効果音MP3
```
